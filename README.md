# Padme desktop demo

Simple desktop implementation of padme core

<img src="/docs/demo.gif" width="300"/>

## Run

```
RUST_LOG=info cargo run mygame.gb
```

## Default controls

* `A` -> A
* `S` -> B
* `Enter` -> Start
* `Tab` -> Select
* `Up` -> Up
* `Down` -> Down
* `Left` -> Left
* `Right` -> Right
